# Git Practice Tasks, Challenges & Assignments

A curated collection of hands-on exercises to build real Git muscle memory.
These tasks progress from beginner to advanced and simulate real workflows used in development teams.

---

## Level 1: Basics (Warm-Up)

| Task No. | Activity                                   |
| -------- | ------------------------------------------ |
| 1        | Initialize a new repo using `git init`     |
| 2        | Create a file, add text, stage it & commit |
| 3        | Modify the same file and commit changes    |
| 4        | Check repo status using `git status`       |
| 5        | Clone any public repo from GitHub          |

Deliverable:
A folder or repo with minimum 3 commits.

---

## Level 2: Branching & Switching

| Task No. | Activity                                         |
| -------- | ------------------------------------------------ |
| 6        | Create a new branch `feature-1`                  |
| 7        | Switch to the branch and add a new file          |
| 8        | Commit the file and merge the branch into `main` |
| 9        | Delete the branch after merging                  |
| 10       | Create a new branch and revert changes           |

Deliverable:
Screenshots/logs of branching operations.

---

## Level 3: Remote + Push/Pull

| Task No. | Activity                                      |
| -------- | --------------------------------------------- |
| 11       | Connect GitHub repo using remote              |
| 12       | Push your local commits to GitHub             |
| 13       | Pull changes from remote on another system    |
| 14       | Create multiple commits and push sequentially |
| 15       | Create & delete a remote branch               |

Deliverable:
Remote GitHub repository link.

---

## Level 4: Merge Conflicts (Must-Practice)

| Task No. | Activity                                               |
| -------- | ------------------------------------------------------ |
| 16       | Create two branches and modify the same line in a file |
| 17       | Merge one into another & generate a conflict           |
| 18       | Resolve conflict manually and commit final output      |

Deliverable:
Final merged file + screenshot of conflict resolution.

---

## Level 5: Stashing & Undo Strategies

| Task No. | Activity                                     |
| -------- | -------------------------------------------- |
| 19       | Make changes but stash instead of commit     |
| 20       | Apply stash back to working tree             |
| 21       | Reset to previous commit using `soft`        |
| 22       | Try a `hard` reset (in a safe practice repo) |

Deliverable:
Stash logs, before/after screenshots.

---

## Level 6: Tags & Releases

| Task No. | Activity                  |
| -------- | ------------------------- |
| 23       | Create lightweight tag    |
| 24       | Push tag to remote        |
| 25       | Checkout code using a tag |

Deliverable:
Tag list output via `git tag`.

---

## Level 7: Advanced + Real-World

| Task No. | Challenge                                           |
| -------- | --------------------------------------------------- |
| 26       | Use `rebase` instead of merge                       |
| 27       | Use `cherry-pick` to move a commit between branches |
| 28       | Create a PR workflow on GitHub                      |
| 29       | Work with `git stash pop` after code changes        |
| 30       | Resolve automatic conflict merging during rebase    |

Deliverable:
PR URL + command logs + summary.
